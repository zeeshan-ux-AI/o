# Zeeshan Portfolio Website

## Overview
A modern, responsive, and professional portfolio website for Zeeshan - a No-Code Web Developer and AI-powered creator from Bangladesh. The website showcases his skills, projects, and services with a stunning developer aesthetic.

## Current State
- Fully functional portfolio website with all sections implemented
- Dark/Light mode toggle with system preference detection
- Smooth animations and transitions
- Mobile-responsive design
- All contact links (WhatsApp, Email, GitHub) are functional

## Tech Stack
- **Frontend**: React with TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Express.js (static file serving)
- **Styling**: Custom CSS animations, glass morphism effects, gradient backgrounds
- **State Management**: React hooks, localStorage for theme persistence

## Project Structure
```

```

## Key Features
1. **Hero Section**: Animated typing effect cycling through roles, profile image with gradient border
2. **About Section**: Professional bio with education and stats cards
3. **Skills Section**: AI platforms and technologies showcase
4. **Projects Section**: Featured work cards with GitHub integration
5. **Services Section**: Four service categories with descriptions
6. **Contact Section**: Direct links to WhatsApp, Email, and GitHub
7. **Theme Toggle**: Persistent dark/light mode preference
